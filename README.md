# python-populous

A Python and pygame reimplementation of the classic Bullfrog god game
*Populous*. Aimed at hobbyists and fans studying the original game on modern
desktops (Windows, macOS, Linux).

## Quick start

```bash
pip install -r requirements.txt
python3 populous.py
```

Requires Python 3 and the dependencies listed in
[requirements.txt](requirements.txt) (`pygame`, `numpy`).
